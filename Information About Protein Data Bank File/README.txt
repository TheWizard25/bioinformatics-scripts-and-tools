Read protein databank (PDB) file and compute following values; 
i) TITLE 
i) Head
ii) RESOLUTION 
iii) HEADER
and other information about a PDB file and store Information into a output Text file using Bio.PDB module

Bio.PDB is a Biopython module that focuses on working with crystal structures of biological macromolecules. 

Biopython should be installed using command

----> pip3 install Biopython (linux users)



Input: PDB format
Output: Text file

Running Command - python3 script.py sample_input.pdb sample_output.txt

(file1.pdp (Protein databank file))
